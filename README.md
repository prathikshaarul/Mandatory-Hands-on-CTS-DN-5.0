# Cognizant Digital Nurture 5.0 (Deep Skilling) вЂ“ Java Full Stack

Welcome to my Cognizant Digital Nurture 5.0 (Deep Skilling) repository.

This repository contains the hands-on exercises, coding assignments, case studies, and learning materials completed as part of the Cognizant Deep Skilling Program. The program focuses on strengthening Java Full Stack development skills through practical implementation of programming concepts, design patterns, databases, Spring Framework, testing, and modern software development practices.

---

## Program Objectives

- Build strong programming fundamentals in Java.
- Understand Object-Oriented Programming principles.
- Learn Design Patterns and their real-world applications.
- Develop database programming skills using SQL and PL/SQL.
- Work with Spring Framework and Spring Boot.
- Implement RESTful Web Services.
- Learn ORM using JPA, Hibernate, and Spring Data JPA.
- Perform Unit Testing using JUnit and Mockito.
- Gain experience with Git and GitHub version control.
- Follow industry-standard coding and project structure.
